# Benchmark Index

This page is the repository's benchmark publication index for the
committed manual pages under `HexManual/Benchmarks/`. It is the staging
point for later rendered benchmark dashboards and release-linked report
snapshots described in [SPEC/benchmarking.md](../../SPEC/benchmarking.md).

The current benchmark reference pages are:

- [`HexArith` benchmark reference](./HexArith.md)
- [`HexPoly` benchmark reference](./HexPoly.md)

These pages document the stable benchmark case names, CSV-style runner
entrypoints, and regeneration hooks that already exist in the committed
Lean sources. Timing results, rendered HTML reports, and release
snapshots remain out of scope for this index page.
